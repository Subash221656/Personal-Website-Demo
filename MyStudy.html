Explore My Study Subjects

Welcome to the fascinating world of my study subjects. Each subject is a unique journey filled with discovery and knowledge.

  1. Data Structure & Algorithm

    Data structures organize and store data, while algorithms are step-by-step instructions for solving problems. Together, they form the backbone of computer science, enabling efficient problem-solving in software development.

  2. Structured Query Language

    SQL is a database language facilitating data management. It enables tasks like querying, updating, and structuring relational databases. Essential for software development, SQL uses commands like SELECT, INSERT, UPDATE, and DELETE.

  3. Web Development & IS

    Web development involves creating and maintaining websites or web applications. It encompasses frontend and backend development, using languages like HTML, CSS, JavaScript, and frameworks for efficient design and functionality.

  4. Basic Python

    Python is a versatile and readable programming language. Known for its simplicity, it's widely used for web development, data analysis, artificial intelligence, and more. Its syntax emphasizes code readability and ease of use.

  5. Cisco Networking

    Cisco Networking entails designing, implementing, and managing computer networks with Cisco devices. It covers routers, switches, and certifications, ensuring expertise in networking technologies for secure and efficient data communication.

  6. Intro to SE

    Software Engineering involves designing, developing, testing, and maintaining software systems. It applies engineering principles to create efficient, scalable, and reliable software solutions, encompassing the entire development lifecycle.

  7. Project & Quality Management

    Project and Quality Management involves planning, executing, and monitoring projects while ensuring high standards. It encompasses effective resource allocation, risk management, and quality assurance for successful project delivery.

  8. Cloud Computing

    Cloud Computing utilizes remote servers to store, manage, and process data over the internet. It provides scalable, on-demand computing resources, fostering efficiency, flexibility, and accessibility for various applications and services.

  9. Information System Security

    Information System Security protects computer systems and data from unauthorized access using measures like encryption and firewalls. It ensures the confidentiality, integrity, and availability of information.
